# Design Doc of "Tank me LoL"

*Core Loop*: Surviving a labyrinth by killing enemies and collecting upgrades
*Goal*: To survive as long as possible
*Challenge*: Enemies actively oppose the player, trying to kill them when possible
*Feedback*: Player always knows what level they are, how long they survived and now many points did they get
*Progress*: More geometric levels, upgrades, new enemies
